# Amigo Secreto

Una aplicación web sencilla para realizar sorteos de amigo secreto.

## 🚀 Características

- Añade participantes al sorteo
- Lista todos los participantes
- Realiza un sorteo aleatorio
- Interfaz intuitiva y fácil de usar

## 🛠️ Instalación

1. Clona el repositorio:
   ```bash
   git clone [https://github.com/tu-usuario/amigo-secreto.git](https://github.com/tu-usuario/amigo-secreto.git)
undefined
Abre el archivo 
index.html
 en tu navegador web.
🎮 Cómo usar
Añadir participantes:
Escribe el nombre en el campo de texto
Haz clic en "Añadir"
Ver participantes:
Los nombres añadidos aparecerán en la lista
Realizar sorteo:
Haz clic en "Sortear amigo"
El resultado se mostrará en la pantalla
🧪 Pruebas
Para probar la aplicación:

Abre el archivo 
index.html
 en tu navegador
Añade al menos 3 nombres de prueba
Verifica que los nombres aparezcan en la lista
Haz clic en "Sortear amigo" y verifica que:
Muestra un nombre aleatorio
El nombre mostrado está en la lista de participantes
No se repiten resultados consecutivos (al menos no frecuentemente)
